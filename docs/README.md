# docs/

Operating manual for this site. Read whichever file matches what you're trying to do.

| File | Read this when… |
|---|---|
| [content-workflow.md](content-workflow.md) | You want to add, edit, or draft a blog post or project. **Start here for everyday changes.** |
| [git-workflow.md](git-workflow.md) | You want to understand the branching model, take a snapshot before a risky change, or revert something. |
| [safeguards.md](safeguards.md) | You want to know which automated checks are running on commits, why, and how to install them on a fresh clone. |
| [setup-guide.md](setup-guide.md) | You're rebuilding this site from scratch (you almost never need this). |
| [setup-log.md](setup-log.md) | You hit an obscure error during setup and want to see how a similar one was solved before. |

For the **architecture** of the site (Astro 5, content collections, Tailwind 4 syntax, theme handling), see `../CLAUDE.md` at the repo root.
